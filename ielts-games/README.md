# IELTS Games

Two standalone HTML games for IELTS vocabulary practice.

## 打打打打字啊啊啊 — Typing Speed Game

**File:** `ielts-typing-game.html`

Words fall from the sky. Type each word correctly before it hits the ground.

- 2,955 IELTS academic vocabulary words
- 3 difficulty levels: Easy (10s) / Medium (7s) / Hard (5s)
- Chinese meaning + English hint displayed on each word card
- Audio pronunciation (local mp3 → Google Dictionary → Web Speech fallback)
- PWA-ready (installable via HTTPS or localhost)

## 同義詞消消消 — Synonym Elimination Game

**File:** `ielts-synonym-game.html`

Choose the word that does NOT share the same meaning as the others.

---

## Usage

Open either HTML file directly in a browser, or serve via a local HTTP server for full PWA support:

```bash
npx serve .
```
